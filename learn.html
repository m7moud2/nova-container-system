Installation

Get Nova running in 60 seconds

Install on macOS

bash 
curl --proto '=https' --tlsv1.2 -sSf https://nova.sh/install | sh

Or install via Homebrew:

bash 
brew install nova

Install on Linux

bash 
curl --proto '=https' --tlsv1.2 -sSf https://nova.sh/install | sh

Or build from source:

bash 
git clone https://github.com/nova-container/core
cd core
cargo build --release
sudo cp target/release/nova /usr/local/bin/

Install on Windows

Install via WSL2 (recommended):

powershell 
wsl --install
wsl
curl --proto '=https' --tlsv1.2 -sSf https://nova.sh/install | sh
